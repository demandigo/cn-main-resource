# cn-main-resource

## 项目简介

本项目是一个通用型 HTML 页面归档与发布仓库。用于集中存放、管理和分发多个独立的 HTML 页面资源，不针对任何特定域名或具体网站。适合用于网页快照保存、静态页面托管、示例页面集合等场景。

## 目录说明

```
cn-main-resource/
├── pages/          # 存放 HTML 页面文件
├── assets/         # 存放页面所需的静态资源（如 CSS、JS、图片等）
├── index.html      # 主入口页面（可选）
└── README.md       # 本说明文件
```

- `pages/`：每个子目录或文件对应一个独立的 HTML 页面。
- `assets/`：按需组织，可包含字体、样式、脚本等公共资源。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开或通过静态托管服务访问。
- 页面内容来自公开来源或自行编写，仅供归档参考。
- 不保证所有页面的时效性、完整性或与原始来源的一致性。
- 归档内容可能不定期更新、补充或移除。
